# 角色档案：朱祁镇

> **姓名**：朱祁镇 
> **年号**：正统（1436–1449）、天顺（1457–1464） 
> **生卒**：1427年－1464年 
> **在位**：正统元年－正统十四年，天顺元年－天顺八年 
> **历史结局**：两度为帝，唯一被俘又复辟的明朝皇帝 
> **原型来源**：《明史·英宗前纪》《明实录》

---

## 一、人物简介

**明英宗朱祁镇**是明朝第六位皇帝，九岁即位，由“三杨”辅政，后宠信宦官**王振**，导致其专权。 
正统十四年（1449年）听信王振之言，**御驾亲征瓦剌**，于**土木堡兵败被俘**，史称“土木之变”。 
其弟朱祁钰即位为景泰帝。 
1457年，通过“夺门之变”复辟，废景泰帝，杀害于谦，是明代最具争议的皇帝之一。

> **游戏定位**：核心 NPC / 被操控的君主 / 主角命运的转折点

## 二、性格分析

| 特质           | 说明                           |
| -------------- | ------------------------------ |
| ✅ **重情义**   | 对王振如父如师，复辟后为其平反 |
| ✅ **轻信人**   | 缺乏政治判断，易被权臣操控     |
| ❌ **好大喜功** | 渴望建功立业，不顾国力发动亲征 |
| ❌ **报复心强** | 复辟后诛杀于谦，贬斥景泰旧臣   |

> **核心矛盾**： 
> 他本可成为仁君，却因宠信宦官、轻率亲征，酿成国难。
